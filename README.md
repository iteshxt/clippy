# Clippy 🌿

**Clippy** is a high-fidelity, nature-inspired online clipboard service. It's built for speed, privacy, and aesthetic delight. Share text, images, gifs, links, and files instantly using unique 4-digit codes—no accounts, no friction.

![Clippy Preview](/public/lizard.png)

## ✨ Features

- 🦎 **Animated Mascot** - Meet the Clippy lizard, who comes to life as you share.
- 🌾 **Procedural Nature UI** - A unique, swaying meadow and rolling hills generated on every reload.
- 🚀 **Zero Friction** - No logins. No accounts. Just paste and share.
- ⏱️ **Auto-Destruct** - Choose expiry from 1m to 24h. Data is wiped automatically via MongoDB TTL.
- 🐳 **Docker-Ready** - Production-optimized Nginx + Node.js + MongoDB stack.
- 🔒 **Secure & Private** - Browser fingerprinting protection and strict 10MB payload limits.

## 🛠️ Tech Stack

- **Frontend:** React 18, Vite, Framer Motion (Animations), CSS Grid/Flexbox.
- **Backend:** Node.js, Express, Multer (File Handling).
- **Database:** MongoDB (TTL Indexes for auto-expiry).
- **Pro Infrastructure:** Nginx (Reverse Proxy), Docker, Docker Compose.
